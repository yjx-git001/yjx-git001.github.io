# YJX 个人主页

一个功能丰富、交互炫酷的个人主页，纯前端实现，无需后端服务。

## 预览

访问地址：https://yjx-git001.github.io

## 功能特性

### 基础功能

- **个人信息展示** - 头像、名字、简介、状态
- **联系方式** - 电话、邮箱、短信（点击查看，再次点击直接拨打/发送）
- **社交链接** - GitHub、微信、QQ、B站
- **技能标签** - 展示技术栈
- **每日一言** - 随机励志名言，支持刷新
- **重要日期倒计时** - 春节、生日倒计时
- **实时天气** - 自动获取当前位置天气
- **访客计数** - 每日访客统计
- **深色模式** - 一键切换明暗主题

### 高级交互

| 功能 | 说明 |
|------|------|
| 鼠标粒子特效 | 鼠标移动时产生彩色粒子拖尾 |
| 3D卡片效果 | 鼠标悬浮卡片时产生3D倾斜 |
| 打字机效果 | 点击简介重播打字动画 |
| 自定义光标 | 炫酷圆形光标，悬浮可点击元素时变大 |
| 滚动进度条 | 页面顶部显示阅读进度 |
| 返回顶部 | 滚动后出现返回顶部按钮 |
| 页面加载动画 | 进入页面时的过渡动画 |
| 卡片闪光效果 | 悬浮时光泽扫过效果 |
| 音乐可视化 | 技能卡片底部跳动的音乐条 |

### 小游戏

- **点击挑战** - 10秒内点击次数挑战，记录最高分

### 隐藏彩蛋

| 触发方式 | 效果 |
|----------|------|
| 双击页面 | 开启/关闭 Matrix 代码雨效果 |
| 连续点击头像5次 | 开启/关闭彩虹背景模式 |
| 输入 ↑↑↓↓←→←→BA | Konami Code 秘籍，触发烟花和页面抖动 |
| 按键盘 `C` 键 | 彩带纸屑特效 |
| 按键盘 `H` 键 | 显示快捷键帮助 |
| 离开页面再回来 | 欢迎回来提示（离开超过10秒） |
| 首次访问 | 彩带庆祝欢迎 |

### 成就系统

连续访问达成里程碑时会弹出成就：

- 🌟 **初露锋芒** - 连续访问3天
- ⭐ **坚持一周** - 连续访问7天
- 🏆 **两周达人** - 连续访问14天
- 👑 **月度冠军** - 连续访问30天
- 💎 **百日传奇** - 连续访问100天

### 个性化设置

点击右上角齿轮图标进入设置面板（密码：`88888888`）

可自定义：
- 头像（支持上传图片）
- 名字、简介、状态
- 联系方式（电话、邮箱）
- 社交链接（GitHub、微信、QQ、B站）
- 技能标签（可增删）
- 生日日期

所有设置保存在浏览器本地存储中。默认个人信息已写入代码，所有设备打开都会显示相同内容。

## 技术栈

- 纯 HTML/CSS/JavaScript
- 无框架依赖
- 响应式设计
- CSS3 动画
- LocalStorage 数据持久化
- Geolocation API（天气定位）
- Open-Meteo API（天气数据）

## 本地运行

直接用浏览器打开 `index.html` 即可。

或使用本地服务器：

```bash
# 使用 Python
python -m http.server 8080

# 使用 Node.js
npx serve
```

## 部署

### GitHub Pages

1. Fork 本仓库
2. 进入仓库 Settings → Pages
3. Source 选择 `main` 分支
4. 保存后等待部署完成

### 其他平台

支持部署到任何静态托管平台：
- Vercel
- Netlify
- Cloudflare Pages

## 自定义修改

### 修改默认配置

编辑 `index.html` 中的 `defaultConfig` 对象：

```javascript
const defaultConfig = {
    name: 'YJX',
    bio: '一个热爱编程的开发者...',
    phone: '18325666364',
    email: '3307852286@qq.com',
    // ...
};
```

### 修改设置密码

搜索 `88888888` 并替换为你想要的密码。

### 修改主题色

搜索 `#667eea` 和 `#764ba2` 替换为你喜欢的渐变色。
